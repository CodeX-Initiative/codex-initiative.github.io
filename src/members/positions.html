Chapter Positions

These are all of the positions we offer for students in each high school chapter.

President


Chapter President, oversees and takes initiative for all major application and managment.

Vice President


Chapter Vice President, assists the president in operations, and takes over tasks the president is unable to complete.

External Secratary


Chapter External Secretary, responsible for all documentation, forms, and other communication work that needs to be completed with external organizations (Sponsor/supporting organizations, PTA, etc)

Internal Secretary


Chapter Internal Secretary, responsible for all documentation, forms, and other communication work that needs to be completed for club members and internal work (Club updates, member and project registeration, chapter registeration, etc).

Treasurer


Chapter Treasurer, assists both secratary in forms and comminication, especially regarding those about sponsorships, donations, monetary expenses.

Main Technical Lead


Chapter Main Technical Lead, responsible for maintaining github repository, pull requests, version management, backend frontend sync, and all other code management and maintanance

Backend Technical Lead


Chapter Backend Technical Lead, responsible for all version control, maintanance, and other tasks related to the backend of the project.

Frontend Technical Lead


Chapter Frontend Technical Lead, responsible for all version control, maintanance, and other tasks related to the Frontend of the project.

Technical Assistant 1


Chapter Technical Assistant, works under/with the main technical assistant. Responsible for update and work presentations.

Technical Assistant 2


Chapter Technical Assistant, works under/with the backend technical assistant. Responsible for update and work presentations.

Technical Assistant 3


Chapter Technical Assistant, works under/with the frontend technical assistant. Responsible for update and work presentations.

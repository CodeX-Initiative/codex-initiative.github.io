SlipSense 1.0

This is a finished project. SlipSense is a developing framework that uses AI models in order to enhance landslide Prediction.

Abstract:

Landslides are a significant geological hazard, causing over 4,000 fatalities and $20 billion in property damage annually. This project aims to predict landslide vulnerability values using advanced deep-learning techniques. A Python-based framework was developed to process diverse remote sensing data, including the Normalized Difference Vegetation Index (NDVI), rainfall, slope, topography, and satellite imagery, to train predictive models with high accuracy and scalability.
Using the USGS Landslide Susceptibility Inventory, multiple CNN models were developed and optimized for each geological feature. The final models achieved accuracies of 99% for satellite imagery, 98% for topography, 97% for slope, and 66% for NDVI. Losses were below 0.2 for all models except NDVI, which had a loss of 1.07. These results demonstrate the models' ability to accurately incorporate diverse terrain and environmental factors to enhance prediction reliability. To converge the CNN predictions, a final XGBoost model was developed, leveraging CNN classification outputs as input features. This ensemble model achieved an impressive accuracy of 98% and a RMSE of 4.9. Validation results consistently matched or exceeded training accuracies, showcasing the models' strong generalization capabilities across various data types.
Future work includes creating a multi-input CNN capable of integrating all data types simultaneously for even greater precision. Additionally, an API will be developed to deliver real-time, globally scalable landslide probability predictions using satellite data. This framework provides a transformative approach to improving disaster risk management and reducing landslide-related impacts worldwide.
